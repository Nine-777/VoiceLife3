### 機能一覧
- ゲストログイン(devise,devise-18n)
- ユーザー登録・編集・削除
- ユーザープロフィール画像設定(carrierwave)
- テキスト投稿・編集・削除
- 音声投稿・視聴(audiojs-rails)
- いいね機能
- いいねランキング
- コメント機能
- キーワード検索(ransack)
- ページネーション(kaminari,bootstrap)


### 技術一覧
- Ruby 3.0.4
- Ruby on Rails 6.1.6
- RSpec(テスト)
- MySQL 5.7
- Docker 2.12.1
- AWS
  - EC2
  - RDS
  - EB
  - Route 53
  - IAM
- HTML
- CSS(SCSS)
